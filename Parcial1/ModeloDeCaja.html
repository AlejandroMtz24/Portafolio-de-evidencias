Pruebas con los modelos de caja
05 de septiembre

Este trabajo consiste en trabajar con los mdodelos de caja y asi aprender mas como funcionan

El objetivo de esta actividad es aprender cómo funciona el modelo de caja en CSS y cómo puedo modificar visualmente los elementos en la página a través de la manipulación de sus márgenes, rellenos y bordes.

Esta práctica me ayudará a comprender mejor cómo los navegadores web renderizan los elementos HTML y cómo ajustar el diseño de una página web usando CSS.
